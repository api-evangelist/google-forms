# Google Forms API (google-forms)
The Google Forms API is a RESTful interface that lets you create, retrieve, and modify forms programmatically. It allows developers to automate form creation, access form responses, and integrate forms into their applications.

**URL:** [Visit APIs.json URL](https://developers.google.com/forms/api)
